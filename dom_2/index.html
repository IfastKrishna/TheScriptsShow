Understanding the DOM in JavaScript

What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing languages like JavaScript to manipulate the content, structure, and style of a document.

DOM Structure

The DOM represents the document as a hierarchical tree structure. Each node in this tree represents a part of the document, such as an element or attribute. The primary types of nodes are:

  • Element Nodes: Represent HTML elements (e.g., <div>, <p>).
  • Text Nodes: Represent the text within elements.
  • Attribute Nodes: Represent the attributes of elements.

Manipulating the DOM with JavaScript

JavaScript can interact with the DOM using various methods, such as:

  • document.getElementById() - Selects an element by its ID.
  • document.querySelector() - Selects the first element that matches a CSS selector.
  • document.createElement() - Creates a new element.
  • element.appendChild() - Adds a new child node to an element.

Example

Here's a simple example of manipulating the DOM:

                
document.getElementById('myElement').innerText = 'Hello, world!';
